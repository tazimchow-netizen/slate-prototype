# Slate Authentication Page

A minimalist login/signup page for **Slate** - a visual workspace application for brainstorming and organizing projects.

## Overview

This authentication page features:
- **Split-screen layout**: Hero section with benefits (left) and authentication form (right)
- **Dual mode**: Toggle between login and signup
- **Google OAuth**: Quick sign-in with Google account
- **Responsive design**: Mobile-first approach with breakpoints at 375px, 768px, 1024px, and 1440px
- **Figma-portable**: All design tokens exported for easy Figma recreation

## Design System Reference

This page follows the design system generated by UI/UX Pro Max skill:
- **Master Design System**: `design-system/visual-board-organizer/MASTER.md`
- **Auth Page Overrides**: `design-system/visual-board-organizer/pages/auth.md`

### Design Principles
- **Style**: Flat Design (minimalist, 2D, bold colors, clean lines)
- **Typography**: Plus Jakarta Sans (friendly, modern, professional)
- **Transitions**: 150-200ms for smooth interactions
- **Accessibility**: WCAG AAA compliant, keyboard navigation, reduced motion support

## Figma Export Guide

### Method 1: Using Figma Plugins (Recommended)

1. **Install a Design Tokens Plugin**:
   - [Tokens Studio for Figma](https://www.figma.com/community/plugin/843461159747178978/Tokens-Studio-for-Figma)
   - [Design Tokens](https://www.figma.com/community/plugin/888356646278934516/Design-Tokens)

2. **Import `design-tokens.json`**:
   - Open your Figma file
   - Launch the Tokens Studio plugin
   - Click "Import" and select `design-tokens.json`
   - The plugin will create Figma variables/styles automatically

3. **Verify Import**:
   - Check color styles match the design system
   - Verify text styles (font family, sizes, weights)
   - Confirm effect styles (shadows)

## Local Development

### Running Locally

Simply open `index.html` in a web browser:

```bash
# Option 1: Direct open
open index.html

# Option 2: Using Python server
python3 -m http.server 8000
# Then visit: http://localhost:8000

# Option 3: Using Node.js
npx serve .
```

## Customization Guide

### Changing Colors

Edit the CSS custom properties in `styles.css`:

```css
:root {
  --color-primary: #000000;   
  --color-cta: #635BFF;       
}
```

Also update `design-tokens.json` to keep them in sync.

### Changing Typography

1. Update the Google Fonts import in `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=Your+Font:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

2. Update the CSS variable in `styles.css`:
```css
:root {
  --font-family: 'Your Font', sans-serif;
}
```

### Adjusting Spacing

Modify the spacing scale in `styles.css`:
```css
:root {
  --space-md: 1rem;   
  --space-lg: 1.5rem;  
}
```
