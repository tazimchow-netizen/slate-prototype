# Slate Authentication Page

A minimalist login/signup page for **Slate** - a visual workspace application for brainstorming and organizing projects.

## Overview

This authentication page features:
- **Split-screen layout**: Hero section with benefits (left) and authentication form (right)
- **Dual mode**: Toggle between login and signup
- **Google OAuth**: Quick sign-in with Google account
- **Responsive design**: Mobile-first approach with breakpoints at 375px, 768px, 1024px, and 1440px
- **Figma-portable**: All design tokens exported for easy Figma recreation

## Design System Reference

This page follows the design system generated by UI/UX Pro Max skill:
- **Master Design System**: `design-system/visual-board-organizer/MASTER.md`
- **Auth Page Overrides**: `design-system/visual-board-organizer/pages/auth.md`

### Design Principles
- **Style**: Flat Design (minimalist, 2D, bold colors, clean lines)
- **Typography**: Plus Jakarta Sans (friendly, modern, professional)
- **Transitions**: 150-200ms for smooth interactions
- **Accessibility**: WCAG AAA compliant, keyboard navigation, reduced motion support
